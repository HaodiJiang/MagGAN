# Solar Image Synthesis with Generative Adversarial Networks

Haodi Jiang et al.

Department of Computer Science, Sam Houston State University
Institute for Space Weather Sciences, New Jersey Institute of Technology

----

## Requirements: 

Tensorflow-GPU 2.10, numpy 1.19.5, astropy 5.1

## Usage: 

Run MagGAN_bxby_prediction.py to produce vector magnetograms of active region for SOHO/MDI in the test data folder of 'test_data'.


## References

- H. Jiang and J. T. L. Wang, "Solar Image Synthesis with Generative Adversarial Networks," 2024 International Conference on Machine Learning and Applications (ICMLA), Miami, FL, USA, 2024, pp. 810-815, doi: 10.1109/ICMLA61862.2024.00116.
